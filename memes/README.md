# memes/

Drop your meme files here. Supported: `.jpg` `.png` `.gif` `.mp4` `.webm`.

The filename must match the value in `config.py` → `GESTURE_MEMES`. With the
default config you'd add:

| gesture   | expected file    |
|-----------|------------------|
| smile     | `smile.jpg`      |
| shock     | `shock.gif`      |
| wink      | `wink.jpg`       |
| thinking  | `thinking.jpg`   |
| shush     | `shush.jpg`      |
| hands_up  | `hands_up.gif`   |

Any gesture with no file yet just shows a text placeholder — the app still
runs, so you can add art incrementally.
